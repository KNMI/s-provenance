  • prov:definition "Any \emph{Collection} of data produced or used by an \emph{Invocation}. Each element gets time-stamped and if not volatile, it can refer to the \emph{url} of its location. It is similar to the Collection entity used by PROV and also by PROV-ONE, however it has been redefined to associate more specific attributes to describe general characteristics of the output data produced; eg. location, format, size. S-PROV assumes for practical reasons that outputs should be always considered like collections of one or more granules."
